# Magic Mod

Идея: добавить в игру _магию_

### Разработка:

- GIGA IDE 
- Microsoft OpenJDK 25
- Minecraft 26
- Fabric mod loader

### Основные ресурсы:

- https://wiki.fabricmc.net/

### Дизайн:

чтобы использовать магию, нужны книга заклинания и магический посох

книга заклинания создаётся на столе заклинаний аналогично столу зачарований: в зависимости от уровня игрока доступны несколько уровней зачарования книги. чем выше уровень зачарования, тем больший уровень заклинания получится, и тем сильнее будет заклинание. книга заклинания не имеет прочности. (чтобы получить заклинание второго уровня, нужно зачаровать книгу первого уровня) (случайность для заклинаний) (комбинации заклинаний)

магический посох - оружие, имеет прочность и дальность применения (атаки). материал посоха влияет на заклинания (добавляет множитель к урону/радиусу/другому численному показателю, например водяной посох увеличивает радиус разлитой воды/увеличивает урон от заморозки или длительность) (разрешает/запрещает использовать заклинания определённых стихий, например через посох воды можно использовать только заклинания воды). имеет несколько уровней, которые зависят от ингредиентов крафта (например водяной посох первого уровня из двух палок и алмаза, второго уровня из двух железных слитков и алмаза, третьего из двух золотых слитков и алмаза). уровень влияет на бонус к заклинанию

заклинания должны быть как боевыми, так и утилитарными. допустим, это будут вода (создание блоков воды), лед (создание льда и заморозка противников), огонь (поджоги и взрывы), земля (создание ям и препятствий из земли, мб какие-то шипы из земли со взрывом). можно сделать заклинания для фермеров: окружить существо забором, быстро вырастить растения
